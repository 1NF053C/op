# get started

```sh
# (first time only) from this dir, create local virtual env
python3 -m venv venv
```

```sh
# (every time you develop) activate local virtual env
source venv/bin/activate
```

```sh
# (only run when venv is active) install dependencies from requirements.txt
pip install -r requirements.txt
```

```sh
# (only run when venv is active) save any installed packages to requirements.txt
pip freeze > requirements.txt
```
